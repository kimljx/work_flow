# 后端说明

- 入口：`backend/app/main.py`
- 数据模型：`backend/app/models.py`
- 认证与令牌：`backend/app/security.py`
- 接口聚合：`backend/app/api.py`

## 主要职责

- 认证与权限校验：登录、刷新令牌、当前用户识别、管理员权限保护
- 任务管理：任务创建、详情、状态变更、锁定、提醒、导入导出
- 通知管理：通知列表、通知详情、收件人送达/已读状态聚合
- 邮件处理：收件配置检测、邮箱基线初始化、手动收件、匹配邮件列表与详情
- 延期审批：成员申请延期、管理员审批、幂等控制与版本控制
- 审计记录：关键管理动作的前后状态留痕

## 服务层职责

- 模板排序与匹配
- 延期审批邮件解析
- 延期审批原子生效
- 通知收件人去重生成
- 最后一个管理员保护

## 注释与维护约定

- 核心函数需说明输入、输出、副作用和异常场景
- 状态流转、权限判断、幂等控制、时间处理需补行内注释
- 变更接口结构时，必须同步更新 `docs/api.md`

测试说明：
- 单元测试位于 `backend/tests`
- 测试库清理脚本位于 `backend/scripts/cleanup_test_db.py`
