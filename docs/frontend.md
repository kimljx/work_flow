# 前端说明

- 技术栈：Vue3 + Vite + Vue Router + Pinia
- 构建目标：`es2019`
- 登录状态存储：access token、refresh token、当前用户资料

## 页面模块

- 管理员看板：展示任务、通知、即时消息触达与重试汇总指标
- 任务管理：展示任务列表、详情、创建编辑、状态色与优先级色；任务列表页提供导入入口，任务详情页可返回列表，编辑页返回任务详情，详情中的通知记录支持进入通知详情
- 任务详情中的延期记录采用审批详情卡片展示，可直接看到申请原因、审批状态、审批渠道、审批时间与审批备注
- 导入导出：支持下载任务 Excel 导入模板，模板内包含字段说明、填写规则与示例数据
- 导入导出页支持直接上传 Excel 模板，并展示成功数量、失败行号和失败原因
- Excel 导入成功后会自动返回任务列表并刷新，同时提示本次成功数和失败数
- 详情页、编辑页、通知详情页等返回入口统一携带来路参数，返回时默认回到进入该页面的来源路由
- 通知中心：展示通知列表与通知详情，详情页包含正文与成员送达明细
- 邮件中心：仅展示已匹配模板的邮件，支持详情、手动收件和下次收件倒计时
- 延期审批：使用卡片式审批工作台，提供统计卡、快捷审批和备注录入
- 成员端：查看我的任务、我的通知及对应详情

管理员端路由：
- 看板
- 任务管理
- 模板管理
- 通知中心
- 延期审批
- 用户管理
- 导入导出
- 审计日志

成员端路由：
- 我的任务
- 我的通知

## 前端公共层说明

- `src/api/http.js`：统一处理接口前缀、令牌注入与 401 自动刷新
- `src/stores/auth.js`：统一管理登录状态、持久化与当前用户资料
- `src/router/index.js`：统一做登录校验与角色路由守卫
- `src/utils/format.js`：统一处理日期、时间与倒计时格式化
- `src/constants/taskUi.js`：统一维护任务状态色与优先级色映射
